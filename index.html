Sprut

Sprut Password Manager

Secure your passwords, protect your privacy.

Easy to use Sprut for iOS

Sprut combines the security of Sprut with a clean intuitive design. Protect yourself using strong passwords — without remembering or typing them.

Automatic Sync

Sprut works with offline and online databases, including automatic two-way synchronization with your cloud storage. Almost any provider that appears in the Files app is supported.

Password Auto Fill

With Auto Fill, you can login to websites and other apps in just a couple of taps.

Convenient

Access your passwords quickly using Face ID or Touch ID sensor. Or disable biometric authentication and use a passcode instead.

Powerful

Custom entry fields, custom icons, TOTP codes. Oh, and you can attach files.

Compatible

Sprut works great with other apps. It supports all the current database formats (kdb, kdbx3, kdbx4) and relies on time-proven implementations of AES, ChaCha20, Twofish, and Argon2 algorithms.

Accessible

Be it VoiceOver or large text on a small screen, accessibility is a first-class citizen in Sprut.

Safe

Everything is backed up: each entry keeps its history, deleted entries end up in Recycle Bin, and each database is copied before saving. (And you can disable any of these.)

Sandboxed

Sprut is an offline app. It will never ask for your server credentials. Instead, it integrates with the iOS Files app, and delegates all the networking to your cloud provider app.

Open Source

Sprut source code is available under the GPL v3 license, so anyone can audit it.

Premium features for advanced users

Multiple Databases

Keep your personal, family and work secrets in different databases, secure and synchronized.

YubiKey Support

For extra security, protect your database with a YubiKey hardware token (NFC and Lightning). Our challenge-response implementation is compatible with SprutXC and Sprut2Android.

Field References

Sprut supports entry field references, so linked entries will refresh automatically when you update the main one.
